DOM stands for Document Object model and is considered The glue between HTML, CSS and JavaScript :
It is an object-oriented representation of a web page, which can be modified with a scripting language such as JavaScript i.e a pages content is stored in the DOM and may be accessed and manipulated (have structure, style and content changed) via JavaScript. In other words, the DOM is a programming interface for HTML and XML documents, and is a hierarchical presentation that browsers use to comprehend a document and object within it on the web Any changes made to the DOM via JavaScript is reflected in what gets shown in the browser, therefore you can dynamically add elements, remove them, move them around, modify attributes on them, set inline CSS styles, and perform all sorts of other types of things. Isn’t that INCREDIBLE! Different browsers have different implementations of the DOM, But every web browser uses some document object model to make web pages accessible via JavaScript.

Because working with the DOM you can allow you to manipulate and view so much of what creates the webpage, i.e get a reference to an element in the DOM, change its text content, apply new styles to it, create new elements and add them to the current element as children, or even delete it altogether, it becomes a great tool to explore alterations to a site in real time, seeing immediately the impact of these changes on a web page.



Arrays and objects are both ways of collecting and storing data in JavaScript:
Objects are a particular data type that can store a collection of properties or characteristics in the form of a key and a value. For example, an object could contain many different properties that pertain to a single person e.g.var client = { Name (key): ‘Emma’ (value), Age (key): 28 (value), Eye color (key): ‘Brown’ (value) ...and so on. }; Accessing/ adding/ changing and removing properties from an object is achieved using either dot or bracket notation. E.g. to get the value of the Age key in the above example I could use client.age Or b) client[‘age’] We could also change the age value: client.age = 82; (Yup, I’m feeling it today!) You can use ‘for...in loops’ to iterate through properties in an object: for (var key in myObject) { console.log(key); // logs keys in myObject console.log(myObject[key]); // logs values in myObject } or by adding the ‘forEach()’ method to Object.keys(): Object.keys(myObject).forEach(function(key) { console.log(key); // logs keys in myObject console.log(myObject[key]); // logs values in myObject }); Arrays Arrays are a particular type of variable that can store a collection of values for that one variable. For example, an array could contain many different values (or names in this example) that pertain to a single person’s 6 cats. Cats in this example are the variable and the 6 different names are the 6 different values for that one variable. E.g. var emmasCats = [‘Bree’, ‘Sam’, ‘Bevuss, ‘Lyall’, ‘Lavender’, ‘Paprika’] To access/ add/ change and remove items from an array we need to use zero-based indexing which means that the first value in the list of values is always 0, then 1,2,3 etc. If I want to access ‘Lavender’ I would write: emmasCats[4]; Items can be added and removed from the beginning or end of an array using the push(), pop(), unshift(), and shift() methods: // push() - Adds item(s) to the end of an array emmasCats.push(‘Poppy’); // pop() - Removes the last item from an array emmasCats.pop(); // unshift() - Adds item(s) to the beginning of an array emmasCats.unshift('Celia', 'Charlie'); // shift() - Removes the first item from an array emmasCats.shift(); You can loop through items in an array in a few ways.There’s the standard for loop: for (var i = 0; i < myArray.length; i++) { console.log(myArray[i]); // logs items in myArray } There’s also the ‘for…of’ loop: for (var item of myArray) { console.log(item); // logs items in myArray } Or we can utilize the ‘forEach()’ method: myArray.forEach(item) { console.log(item); // logs items in myArray });



What I think the difference between Emotional Intelligence and IQ is

My understanding is that IQ is a measure of a person's ability to reason using abstract logic and rationale. It will determine a person's speed and accuracy as it comes to solving complex cognitive tasks and relies on functions like attention, the formation of knowledge, memory and working memory, judgment and evaluation, reasoning and "computation", problem solving and decision making, comprehension and production of language.

It is different to emotional intelligence both in the processes it more heavily relies on, e.g. learning, abstract and spatial thinking compared to perception/evaluation and expression. IQ seems to rely on more tangible, inert and measurable elements while emotional intelligence relies on largely unseen, animated and dynamic elements that exist within and between people



Why I think Emotional Intelligence is important

Emotional intelligence is important as it allows for greater understanding, ability and process and therefore choice as it comes to experiencing and expressing emotions throughout life. In the workplace it can nurture collaboration, service, initiative, creativity and leadership. In the home it can do the same, it can enhance and nourish relationships and I believe it can impact overall wellness. It can often inform us of the reasoning and motivations for desiring certain things and making certain choices.

I think the power of understanding what determines our choices is incredibly empowering as it grants us the opportunity to effectively support the changes we want to see in our lives.
